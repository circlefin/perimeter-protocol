# perimeter-core

This repository contains the contracts for the Perimeter protocol.

## Getting started

```sh
npm install
```

## Running a node

```sh
npx hardhat node
npx hardhat run scripts/deploy.ts
```

## Testing

```sh
npm test

# or

npx hardhat test
```

### Gas usage reports

```sh
npm run test:gas

# or

REPORT_GAS=true npx hardhat test
```

### Code Coverage

```sh
npm run test:coverage

# or

npx hardhat coverage
```
